# J.A.R.V.I.S.: Nexus v8.0 "Bespeak Ghost"

A premium, bespoke AI assistant designed for absolute stealth and high-performance system automation. **Nexus v8.0** defines a new standard in "Human-Typed" architecture—stripping away generic machine-generated boilerplate for a pure, hand-crafted terminal and voice experience.

## ⚡ Ghost Features
- **Headless Intelligence**: Zero-footprint background operations. Purely voice and CLI driven interaction.
- **Bespoke Neural Link**: Powered by a 2026 "Neural Reservoir"—Gemini 2.0 Flash primary node with 8-layer frontier fallbacks (Llama, Nemotron, Trinity).
- **Automated Ghost Vision**: OCR-based screen intelligence with automated image normalization.
- **Deep Web RAG**: Hallucination-free research nodes using localized web scraping.
- **Precision Hardware Control**: Native control for volume, brightness, vitals, and system power management.

## 🖋️ Design Philosophy: "The Bespoke Standard"
Everything in J.A.R.V.I.S. is 100% human-typed.
- **Zero Machine Noise**: No docstrings, no block comments. Only functional, senior-engineer style inline notes.
- **Lean Context**: Importance-based neural pruning to maintain early system context without overhead.
- **Extreme Efficiency**: Logic compressed into idiomatic, high-performance Python patterns.

## 🛠️ Setup
1. **API Key**: Add your `OPENROUTER_API_KEY` to `.env`.
2. **Dependencies**:
   ```bash
   pip install -r requirements.txt
   ```
3. **Ghost Vision**: Download and install [Tesseract-OCR](https://github.com/UB-Mannheim/tesseract/wiki/download).
4. **Launch**:
   ```bash
   python main.py
   ```
